You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /guess?num1=2&num2=7 --you chose two numbers to try and guess a number 1-10 num1 and num2 can be any number of your choosing
  • /palindrome?str1=hello&str2=racecar --you choose two strings to check if they are palindromes.
